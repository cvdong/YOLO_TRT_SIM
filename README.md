# YOLO_TRT_SIM

 一套代码同时支持 V5,V6,V7,V8 TRT推理 ™️，前后处理均由CUDA核函数实现 :rocket:

 该REPO功能描述：
 - 支持onnx转TRT FP32 FP16 INT8 engine;
 - 支持动态batch 推理；
 - 支持image和video 推理；
 - 支持多路多线程并行推理；
 - 仅依赖opencv和tensorrt;
 - 支持YOLOV5 V6 V7 V8 推理；
 - 友好的封装格式，便于学习

### MY ENVIRONMENT

- cuda 11.7
- cudnn 8.4
- opencv 4.6
- tensorrt 8.4
